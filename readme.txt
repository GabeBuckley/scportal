************************************************************
* Security Portal Clickable Demo - Instructions
************************************************************

IMPORTANT NOTE: This is windows only. If you try to run this on a mac it will not work.

Method:
---------------------------------------------------------------
Unzip the file into a folder (location is unimportant)

Browse to the folder you just unzipped the site into and:
    
    - From the command line:
        Type, "runme.bat" and press ENTER
        
    - From windows explorer:
        Double-click the runme.bat file
        
A Browser window should open to http://localhost:8000/

The clickable demo should open in this window.



Cleanup:
-----------------------------------------------------------------
To kill the web server, press Ctrl+C in the server dialog window and answer "Y" when prompted
